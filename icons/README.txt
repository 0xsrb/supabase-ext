ICONS FOLDER
============

This folder needs 3 PNG files for the Chrome Extension to work:

Required Files:
  - icon16.png  (16x16 pixels)
  - icon48.png  (48x48 pixels)
  - icon128.png (128x128 pixels)

HOW TO CREATE THEM:
===================

Method 1: Use the Icon Generator (EASIEST)
------------------------------------------
1. Go back to the parent folder
2. Open "create-icons.html" in your browser
3. Click "Generate Extension Icons"
4. Three files will download
5. Move them here

Method 2: Use Any Image Editor
-------------------------------
1. Create 3 PNG images in sizes 16x16, 48x48, 128x128
2. Use any icon/logo you want
3. Name them icon16.png, icon48.png, icon128.png
4. Save them in this folder

Method 3: Download from Icon Site
----------------------------------
1. Visit https://www.favicon-generator.org/
2. Upload any image
3. Download the generated icons
4. Rename to match: icon16.png, icon48.png, icon128.png
5. Move here

CURRENT STATUS:
===============
This folder is currently empty.
The extension will NOT load until these 3 files are added.

Once you add the icons, you can load the extension in Chrome!
